# 刻意练习

练习方法不对，越练越错
专注于练习20%的核心技能
让大脑跟随身体，将技能变成习惯
为什么你会在淋浴、开车和刷牙时脑洞打开
集中时间和精力设定练习目标
练习最擅长的，放大优势效应
专项练习宁多勿少，实战演练宁少勿多
反复练习正确动作，加强大脑记忆
